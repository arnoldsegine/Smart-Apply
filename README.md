# ⚡ PM Copilot — AI Job Assistant for Senior PMs

An AI-powered job application assistant built for Senior Product Managers, AI PMs, and Product Owners.

## Features
- ⚡ **Auto Apply** — Paste a JD, get a full application packet instantly
- ◈ **Resume Tailor** — Tailor your resume to any role
- ◆ **Cover Letter** — Executive-level cover letters in seconds
- ◉ **Job Fit Score** — Honest gap analysis scored out of 10
- ◎ **Interview Prep** — Practice with AI feedback

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```
Open http://localhost:5173

### 3. Build for production
```bash
npm run build
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
3. Leave all settings as default → Deploy
4. Add your custom domain in Settings → Domains

## Tech Stack
- React 18
